## üìñ How to Use

Once you have the application running, here‚Äôs how you can get started:

1.  **Sign Up or Log In:**
    Begin by creating a new account or logging in with your existing credentials. All your work is securely tied to your account.

2.  **Navigate Your Dashboard:**
    After logging in, you'll be greeted by the main dashboard. This is your central hub for accessing the Lesson Planner, Exam Generator, and viewing reports.

3.  **Create with the Lesson Planner:**
    Use the sidebar to navigate to the **Lesson Planner**. Here, you can create, view, and manage structured lesson plans tailored to your curriculum.

4.  **Generate Exams with Ease:**
    Head to the **Exam Questions** section to build custom exams. You can add questions, manage them, and prepare them for your students. The tool supports exporting to formats like DOCX and PDF.

## ÔøΩ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Node.js (version 18.x or higher recommended) and a package manager like npm or pnpm installed.

### Installation

1.  Clone the repository:
    ```sh
    git clone https://github.com/[your-username]/eduforge-frontend.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd eduforge-frontend
    ```
3.  Install the dependencies:
    ```sh
    npm install
    # or
    pnpm install
    ```
4.  Create a `.env` file in the root of the project and add the necessary environment variables. You can use `.env.example` as a template.
    ```
    VITE_API_BASE_URL=http://localhost:8080/api
    ```
